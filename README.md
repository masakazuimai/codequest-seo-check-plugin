# CodeQuest SEO Check - WordPress Plugin

WordPress管理画面からワンクリックでサイトのSEOスコアを診断できるプラグインです。

## 機能

- 100点満点のSEOスコア表示
- 4カテゴリ評価（構造化データ・基本SEO・コンテンツ・技術SEO）
- 個別チェック項目の合否・改善提案
- 日英多言語対応（WordPress言語設定に連動）
- APIキー連携（有料プランで回数枠拡大）

## インストール

1. [Releases](https://github.com/masakazuimai/codequest-seo-check-plugin/releases)からZIPをダウンロード
2. WordPress管理画面 → プラグイン → 新規追加 → アップロード
3. 有効化して左メニューの「SEO Check」から診断

## APIキー設定（任意）

1. [seo.codequest.work](https://seo.codequest.work)でアカウント登録
2. ダッシュボード → APIキー管理でキーを発行
3. WordPress管理画面 → SEO Check → 設定 → APIキーを入力

APIキーなしでも3回まで無料で診断できます。

## 動作要件

- WordPress 6.0以上
- PHP 7.4以上
